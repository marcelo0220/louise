# Louise Shop — lancer en local

Instructions rapides pour exécuter le projet dans le navigateur :

1. Ouvrez un terminal dans le dossier du projet (d:\shop louise)
2. Installez les dépendances :

```bash
npm install
```

3. Lancez le serveur de développement Vite :

```bash
npm run dev
```

4. Ouvrez le navigateur à l'adresse indiquée (par défaut http://localhost:5173)

Remarques :
- Le composant principal est le fichier `louise-shop.jsx` à la racine.
- Si vous préférez ne pas installer de dépendances, je peux aussi générer une version sans bundler (Babel standalone), mais `lucide-react` nécessite un bundler pour fonctionner proprement.
